## 9. theastrologer-api
Un-official REST API to view and send horoscope as SMS from
[theastrologer.com](http://theastrologer.com) using
[theastrologer](https://pypi.python.org/pypi/theastrologer/)
lilbrary, Flask framework and Twilio API.
